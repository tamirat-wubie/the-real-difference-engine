# Edge Case Rules

## 1. Unrelated Things

Unrelated things can be compared only if the lens is clear.

Bad:

Elephant vs Democracy

Better:

Elephant vs Democracy through memory

## 2. Physical vs Abstract

Allowed when the lens connects them.

Example:

Fire vs Language through transformation.

Final truth:

Fire transforms matter. Language transforms meaning.

## 3. Fictional Comparisons

Always specify version.

Example:

MCU Hulk vs MCU Thanos  
Comics Hulk vs Comics Thanos

## 4. Controversial Topics

Use evidence labels.

Separate:

- fact
- inference
- speculation
- judgment

## 5. Human Groups

Do not rank human identity groups as superior or inferior.

Allowed:

Compare systems, institutions, incentives, behaviors, strategies, or historical conditions.

Not allowed:

Claims that one protected group is inherently better or worse than another.

## 6. Health, Legal, Financial Topics

Keep content educational.

Avoid direct personalized instruction unless context and qualifications support it.

## 7. False Balance

Do not force symmetry when evidence is asymmetric.

Example:

Truth vs Lie should not pretend both are equally valid.

Better lens:

Truth vs Lie through trust survival.

Final truth:

A lie can move faster, but truth can carry reality longer.

## 8. Copyright Risk

Use original narration, original visuals, public-domain assets, licensed assets, or minimal transformative reference where legally appropriate.

Do not build the channel on copied clips.
