# Platform Overview

## Name

**The Real Difference Engine**

## Public Brand

**The Real Difference**

## Purpose

The platform compares any two things through a clear lens, reduces each side to first principles, maps causal chains, and reveals the hidden difference.

## Public Promise

I reveal the real difference between things people compare badly.

## Why It Exists

Most comparisons stay at the surface.

They ask:

- Which is better?
- Which is stronger?
- Which wins?
- Which is more popular?

The Real Difference Engine asks:

- What is each thing at the root?
- What lens makes the comparison meaningful?
- What causal chain explains each side?
- What hidden similarity connects them?
- What hidden difference separates them?
- When does A win?
- When does B win?
- What useful truth appears?

## Core Law

No lens, no useful truth.

## Core Method

Define -> reduce -> map -> compare -> reveal -> judge -> compress.
