# Visual System

## Core Visual Structure

Use six frames for short videos.

1. A vs B
2. Lens
3. A root
4. B root
5. Hidden difference
6. Final truth

## Example

Frame 1:

DISCIPLINE vs MOTIVATION  
People compare this wrong.

Frame 2:

Real comparison:  
Behavior continuity

Frame 3:

Motivation:  
feeling -> energy -> action

Frame 4:

Discipline:  
rule -> repetition -> continuity

Frame 5:

Hidden difference:  
Motivation depends on feeling.  
Discipline survives feeling.

Frame 6:

Final truth:  
Motivation starts motion.  
Discipline preserves motion.

## Thumbnail Rules

Use 3-6 words where possible.

Good:

- Attention Is Not Income
- Intensity vs Persistence
- Brand Is Not Reputation

Avoid crowded text.
