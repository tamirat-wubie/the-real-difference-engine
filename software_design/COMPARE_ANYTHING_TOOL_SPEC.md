# Compare Anything Tool Spec

## Purpose

User enters two things plus a lens and receives a structured comparison output.

## Inputs

- A
- B
- primary lens
- secondary lens
- context
- depth
- output type

## Outputs

- first-principle roots
- causal chains
- hidden similarity
- hidden difference
- failure modes
- conditional verdict
- final memorable line
- short video script
- long video outline
- newsletter draft
- custom report draft
- JSON object

## Algorithm

Receive A, B, lens, context  
-> validate lens  
-> define A and B  
-> reduce to first principles  
-> generate causal chains  
-> detect hidden similarity  
-> detect hidden difference  
-> detect flips and failure modes  
-> produce conditional judgment  
-> compress final line  
-> package output by format
