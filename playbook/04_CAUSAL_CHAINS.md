# Chapter 4 - Causal Chains

A causal chain explains how something becomes what it does.

It answers:

- What starts it?
- What powers it?
- How does it move?
- What does it create?
- What limits it?
- How does it fail?

Template:

`origin -> driver -> behavior -> result -> constraint -> failure`

Example:

Motivation:

`feeling -> energy -> action -> emotional decay -> inconsistency`

Discipline:

`rule -> repetition -> habit -> identity -> compounding`

Do not compare descriptions. Compare causal chains.
