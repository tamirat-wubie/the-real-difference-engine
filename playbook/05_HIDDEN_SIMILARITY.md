# Chapter 5 - Hidden Similarity

A hidden similarity is the shared structure beneath two different things.

It answers:

**What do A and B secretly have in common?**

Example:

Fire and water look like opposites.

Hidden similarity:

**Both transform reality.**

Without hidden similarity, the comparison feels random.

With hidden similarity, the comparison becomes meaningful.

Types:

1. Shared function
2. Shared behavior
3. Shared dependency
4. Shared transformation
5. Shared failure pressure
